## 🚀 启动建议

### 完整环境启动步骤：

1. **安装Docker和Docker Compose**
   ```bash
   # 确保Docker已安装并运行
   docker --version
   docker-compose --version
   ```

2. **启动基础设施服务**
   ```bash
   # 启动所有依赖服务
   docker-compose up -d postgres redis etcd prometheus grafana jaeger
   ```

3. **等待服务就绪**
   ```bash
   # 等待数据库初始化完成
   sleep 30
   ```

4. **启动微服务**
   ```bash
   # 启动用户服务
   ./bin/user-service.exe &
   
   # 启动API网关
   ./bin/api-gateway.exe &
   ```

5. **验证服务**
   ```bash
   # 运行测试脚本
   go run test_project.go
   ```
